## 📖 Overview

**Learning English** is a web-based online school platform designed to help learners improve their English language skills through interactive lessons, quizzes, and educational resources.

The project was developed as part of an **Internet Programming** coursework assignment to demonstrate the practical application of frontend web development concepts using **HTML5, CSS3, and JavaScript**.

The website focuses on creating a clean, responsive, and user-friendly learning environment while implementing core Internet Programming principles such as semantic HTML, DOM manipulation, form validation, client-side scripting, and responsive web design.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Reigns-B/learning-english.git
```

### Navigate into the project

```bash
cd Online-School
```

### Open the website

Simply open **index.html** in your preferred web browser.

No additional installation is required.
